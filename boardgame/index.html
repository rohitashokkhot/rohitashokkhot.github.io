üé® Design Philosophy

Visual Hierarchy: Problem tiles use urgent red gradients to convey degradation, while solution tiles use nature-inspired colors that correspond to their domain (green for ecology, blue for technology, orange for community).

Icon Language: Each tile uses 4 clear icons that immediately communicate key concepts. Icons progress from problem indicators on degenerated tiles to solution benefits on regenerative tiles.

Connection System: The bottom section of each tile shows which other solutions it connects with, plus synergy strength dots. This replaces complex magnetic systems with clear visual information.

Level Progression: Badge system shows regenerative impact from Level 1 (degenerated) through Level 6 (fully regenerative), helping players understand the progression toward true regeneration.

üìã Production Considerations

Size: Recommend 3.5" x 5" tiles for comfortable handling and clear text readability

Materials: Heavy cardstock (300gsm) with matte finish to reduce glare and provide durability

Weights: Embed metal washers in sealed compartments within tile backing for balance scale functionality

Colors: Use high-contrast color schemes to ensure accessibility for colorblind players

Text: Minimum 10pt font size for descriptions, with clear sans-serif typeface
